Аметхан Шүгыла

15 мамыр, 2007 жылы Талдықорған қаласында туылған

Менің сүйікті ісімдерім:

  • Гитара ойнағанды жақсы көремін
  • Музыка тыңдағанды жақсы көремін
  • Тауларға шыққанды ұнатамын

Менің мақсатым:

Қазіргі уақытта Абай атындағы Қазақ ұлттық педагогикалық университетінде физика мамандығында 1 курс студентімін.

Портфолио

Сайтты жасау бойынша қадамдар:

  • 1-қадам: HTML құрылымын жасаңыз (сайттың негізгі беті).
    Мысалы, менің туралы мәлімет, сүйікті ісімдерім және цитаталар.
  • 2-қадам: CSS арқылы стильдерді анықтаңыз (шрифт, түстер, фон).
    Мен бордовый және жасыл түстерді таңдадым.
  • 3-қадам: Сайттың құрылымын тексеріңіз және дұрыс көрсетілуін қамтамасыз етіңіз.
  • 4-қадам: GitHub-та репозиторий жасаңыз, файлдарыңызды сонда жүктеңіз.
  • 5-қадам: GitHub Pages арқылы сайтты онлайнға шығарыңыз.

GitHub арқылы сайтты қалай орналастырдым:

  1. GitHub-та жаңа репозиторий жасаңыз.
  2. Сайттың барлық файлдарын осы репозиторийге жүктеңіз.
  3. GitHub Pages бөлімінде "main" немесе "master" branch-ті таңдаңыз.
  4. Сайттың URL мекенжайы пайда болады (мысалы, https://username.github.io/repository-name/).
